# BeltramiNet

This repository will collect codes accompanying the publication:
Sarah J Hamilton, Asko Hänninen, Andreas Hauptmann, Ville Kolehmainen (2019). Beltrami-net: domain independent deep D-bar learning for absolute imaging with electrical impedance tomography (a-EIT).
https://iopscience.iop.org/article/10.1088/1361-6579/ab21b2

Scripts will need Python, and TensorFlow

Note: Basic codes for training and evaluation. Training data will not be provided.
Some network files can be shared on reasonable request, please contact: a.hauptmann@ucl.ac.uk

- 04 July 2019: First commit, python codes for evaluation with 2 KIT4 examples
